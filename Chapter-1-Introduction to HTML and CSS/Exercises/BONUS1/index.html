HTML & CSS Quick-Reference Guide

A comprehensive guide to essential HTML tags and CSS properties

HTML Fundamentals

HTML (HyperText Markup Language) is the standard language for creating web pages. Here are some essential tags you'll use frequently:

Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

1. Headings <h1> to <h6>

Used for titles and subtitles with h1 being the most important and h6 the least.

<h1>Main Page Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>

2. Paragraphs <p>

Used for blocks of text content.

<p>This is a paragraph of text. It can contain multiple sentences and will be displayed as a block element with space before and after.</p>

3. Links <a>

Creates hyperlinks to other pages or resources.

<a href="https://example.com">Visit Example.com</a>
<a href="contact.html">Contact Page</a>
<a href="#section-id">Jump to Section</a>

4. Images <img>

Embeds images in your web page.

<img src="image.jpg" alt="Description of image">
Always include the alt attribute for accessibility.

5. Lists <ul>, <ol>, <li>

Create unordered (bulleted) or ordered (numbered) lists.

<!-- Unordered list -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Ordered list -->
<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

6. Divisions <div>

A container for grouping and styling content.

<div class="container">
    <h2>Section Title</h2>
    <p>Content goes here...</p>
</div>

7. Spans <span>

An inline container for styling part of a text.

<p>This is <span style="color: red;">important</span> text.</p>

8. Code Blocks <code> and <pre>

For displaying code snippets and formatted text.

<pre>
    <code>
    function greet() {
        console.log("Hello, world!");
    }
    </code>
</pre>

9. Tables <table>

Used to display tabular data.

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>

10. Forms <form>, <input>

Collects user input.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>

11. Semantic Elements

Provide meaning to your page structure.

<header>Page header content</header>
<nav>Navigation links</nav>
<main>Main content area</main>
<article>Self-contained content</article>
<section>Grouped content</section>
<footer>Page footer content</footer>

CSS Essentials

CSS (Cascading Style Sheets) is used to style and layout your web pages. Here are some common properties:

Property Description Example
color Sets text color color: #3498db;
background-color Sets background color background-color: #f1f1f1;
font-family Sets the typeface font-family: Arial, sans-serif;
font-size Sets text size font-size: 16px;
margin Sets space outside elements margin: 10px 20px;
padding Sets space inside elements padding: 15px;
border Sets element borders border: 1px solid #ddd;
display Sets how element is displayed display: flex;
position Sets positioning method position: relative;
width/height Sets dimensions width: 100%;

CSS Selectors

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.container {
    max-width: 1200px;
    margin: 0 auto;
}

/* ID selector */
#header {
    background-color: black;
    color: white;
}

/* Descendant selector */
nav a {
    text-decoration: none;
}

/* Pseudo-class */
a:hover {
    color: red;
}
